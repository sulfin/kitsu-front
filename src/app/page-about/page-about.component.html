DESCRIPTION :

ce projet recrée les fonctionnalités principales d'un site de centralisation de tous ce qui porte sur la culture anime japonaise

FONCTIONNALITES :

  • page accueil avec les derniers épisodes sorties et les animés populaire
  • page sorties qui affiches une liste paginée des derniers épisodes sorties
  • page anime qui affiche la liste d'anime par ordre alphabétique avec liens à la page de description pour chacun
  • page saisons permettant de filtré les contenus sortie par année et saison

RESSOURCES :

  • documentation angular
  • documentation bootstrap
  • mdn
  • API kitsu

Projet réalisé dans le cadre de l'évaluation d'angular de janvier 2023 sous angular 14, mis-en-page avec bootstrap.
